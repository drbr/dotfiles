# Git in the Zone

Git is historically notorious for being somewhat hard to understand and
use. This is my attempt at a tutorial.

[![XKCD comic introducing Git: "How do we use it?" "No idea; just memorize these shell
commands…"](https://imgs.xkcd.com/comics/git.png)](https://xkcd.com/1597)

After having used Git for many years and helping my coworkers get unstuck, I started thinking about
how I would teach it. I first learned Git while working at Amazon in the early 2010s, during which
time the company was starting to use Git internally. The company provided us with training sessions
that left me with a solid understanding of how all the different concepts and commands fit together,
and ever since then I've been the friend that people call when they need help.

This tutorial was originally conceived as a tech talk with a live demo. I've given the talk a couple
times to various groups, and the info here is some combination of an outline and a script. Although
these written materials should be understandable on their own, they don't include super-detailed
step-by-step instructions or screenshots because they're intended to be consumed along with a
demonstration.
